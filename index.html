All Git Commands change

learn all git commands

  1. create new branch from master: git branch branchName
  2. switch branch: git checkout branchName
  3. status: git status
  4. Add the files: git add .(if in the current branch)
  5. commiting the code: git commit -m "message"
  6. pushing the code: git push origin banchname
  7. Before start working in any branch: git pull origin branchname from where you want the code
