learn all git commands

    process

  1. create new branch from master: git branch branchName
  2. switch branch: git checkout branchName
  3. status: git status
  4. Add the files: git add .(if in the current branch)
  5. commiting the code: git commit -m "message"
  6. pushing the code for the first time when u create a new branch: git push origin -u banchname
  7. Before start working in any branch: git pull origin branchname from where you want the code, ideal is to take from master bcz master is have code up date from many feature branches git pull origin master
  8. when work done in feature module then delete branches
  9. git branch -d branchname
  10. git push origin --delete branchname
